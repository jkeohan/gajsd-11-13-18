JavaScript Object Notation (JSON)

JSON: What is it?

• JSON: JavaScript Object Notation

• JSON !== JavaScript

• Text-based data format that is based on JS

• JSON is language-independent; Ruby, Python, etc. can all parse/generate JSON

• Successor to XML

• Easy for humans to read; easy for machines to parse

• Specified by Douglas Crockford

JSON: What's it used for?

• JSON is used as a data-exchange format between programs

• Ex: sending data from the frontend (JavaScript), to a Python-powered backend, and vice-versa

• Has strict conventions (unlike JS):

⇒ Property names must be double-quoted strings: "name" vs name

⇒ All strings must be double-quoted: "name" vs 'name'

⇒ No in-line comments

⇒ Trailing commas are forbidden

JSON: JavaScript's JSON Object

• Like the Math object, JavaScript has a lib for working with JSON: JSON

• JSON.stringify() ⇒ Transforms JavaScript to JSON


  var jsonObject = JSON.stringify({ name: 'chandler' });
  // "{"name":"chandler"}"

  typeof jsonObject
  // string

• JSON.parse() ⇒ Transforms JSON to JavaScript


  var javaScriptObject = JSON.parse(jsonObject);
  // { name: "chandler" }

  typeof javaScriptObject
  // "object"
